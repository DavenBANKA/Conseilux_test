CEFR Levels Explained

Understanding the Common European Framework of Reference for Languages

A1 - A2

Basic User

Elementary proficiency - Can handle simple, everyday situations

  • Understand everyday expressions
  • Introduce yourself and others
  • Ask and answer simple questions
  • Interact in a simple way
  • Write short, simple postcards
  • Fill in forms with personal details
B1 - B2

Independent User

Intermediate proficiency - Can handle most situations while traveling

  • Understand main points of clear input
  • Deal with travel situations
  • Produce connected text
  • Describe experiences and events
  • Understand complex texts
  • Interact with native speakers fluently
C1 - C2

Proficient User

Advanced proficiency - Can use language flexibly and effectively

  • Understand virtually everything
  • Express yourself fluently
  • Use language flexibly
  • Produce clear, well-structured text
  • Understand implicit meaning
  • Express ideas precisely
